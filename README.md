# Conway's Game of Life
This is a cellular simulation, with the following rules
1. An living cell continues to live if it has 2-3 neighbouring living cells
2. A living cell dies if it has less than 1 neighbours (as if by solitude)
3. A living cell dies if it has more than 3 neighbours (as if by overpopulation)
4. If there is space which is surrounded by exactly 3 living cell, it gets populated by a living cell

Initially you'll only see empty space. To populate the grid, pause the simulation and click the space using your mouse.
